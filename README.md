## Service Discovery

Aplikasi yang menyimpan database berisi semua nama aplikasi dan lokasinya (host dan port). Ada beberapa implementasi discovery services, antara lain:
- Eureka
- Consul


## Run Services
```
mvn spring-boot:run
```
